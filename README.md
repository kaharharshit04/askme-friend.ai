# 🤖 askme-friend

**askme-friend** is a simple friendly AI chatbot built using **LangChain**, **FastAPI**, and **React.js**, powered by **Gemini LLM**.
The project demonstrates how a modern AI chatbot can be built using a full-stack architecture with a Python backend and a React frontend.

The chatbot provides conversational responses to user queries and showcases how Large Language Models (LLMs) can be integrated into real applications using modern AI frameworks.

---

# ✨ Features

* 💬 Friendly conversational chatbot
* ⚡ Fast backend using FastAPI
* 🧠 LLM powered responses using Gemini
* 🔗 LangChain for LLM orchestration
* 🎨 Clean and simple chat UI built with React

# ⚙️ Installation

## 1. Clone the Repository

git clone https://github.com/kaharharshit04/askme-friend.ai

cd askme-friend

---

# 2. Backend Setup

cd backend

pip install -r requirements.txt

Create a `.env` file and add your Gemini API key:

GEMINI_API_KEY=your_api_key

Start the backend server:

uvicorn main:app --reload

Backend will run at:

http://localhost:8000

---

# 3. Frontend Setup

cd frontend

npm install

npm start

Frontend will run at:

http://localhost:3000

---

# 🚀 Usage

1. Start the backend server
